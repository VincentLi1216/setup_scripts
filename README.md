# Step 1
Cloning the project
```shell
git clone https://github.com/VincentLi1216/setup_scripts.git
cd setup_scripts
```

# Step2
Making the scripts executable.
```shell
chmod +x setup.sh
```

# Step3
Run the Scripts.
```shell
sh setup.sh
```

# Step4(Optional)
Setting up the zsh, and Oh-My-ZSH.
```shell
sh setup-zsh.sh
```
